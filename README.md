# GRUB-Themes

to install run install.sh in theme folder via terminal

```bash
./install.sh
```

### To further customise GRUB install GRUB-Customizer

```bash
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt update
sudo apt install grub-customizer
```
